# Security Incident Mapping (SIM)

## Project Description
Security Incident Mapping (SIM) is a web application developed to improve campus security by enabling students, staff, landlords, and residents to report security incidents quickly and efficiently.

## Objectives
- Improve campus security
- Enable easy reporting of incidents
- Store incident records
- Help security personnel respond faster

## Features
- Report security incidents
- Submit personal details
- Record incident location
- Display success message after submission

## Technologies Used
- Python
- Flask
- HTML
- CSS
